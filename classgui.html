Detailed Description

This class encapsulates the lcd drawing code used to display the various user interfaces on the lcd.

◆ level_view()

void gui::level_view ( float  in_level,
float  out_level,
jack_client::sample_t  sample_array[7][BUFFER_LENGTH] 
)

Draw peak volume level bars for each position in the effect chain over the last BUFFER_LENGTH samples, also show the current input and output gain.

sample_array[0] = input multiplied by input level
sample_array[1] = level after 1st effect
sample_array[2] = level after 2nd effect
sample_array[3] = level after 3rd effect
sample_array[4] = level after 4th effect
sample_array[5] = level after 5th effect
sample_array[6] = output multiplied by output level
Parameters
in_levelInput gain level (0<=in_level<=1.0).
out_levelOutput gain level (0<=out_level<=1.0).
sample_arrayArray of previous samples for each position in the effect chain.
