Detailed Description

This class encapsulates a simple interface for a GPIO input using the Linux gpiochip kernel interface defined in /usr/include/linux/gpio.h.

Example:

#include "gpio.h"
#include <unistd.h> //sleep()
#include <iostream> //std::cout
int main(void)
{
//Read input every second
gpioin in("/dev/gpiochip0", 42);
while(1)
{
if(in)
std::cout << "HIGH\n";
else
std::cout << "LOW\n";
sleep(1);
}
}
