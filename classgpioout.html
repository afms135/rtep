Detailed Description

This class encapsulates a simple interface for a GPIO output using the Linux gpiochip kernel interface defined in /usr/include/linux/gpio.h.

Example:

#include "gpio.h"
#include <unistd.h> //sleep()
int main(void)
{
//Toggle output every second
gpioout out("/dev/gpiochip0", 42, 0, "Test Output");
while(1)
{
out = 1;
sleep(1);
out = 0;
sleep(1);
}
}
