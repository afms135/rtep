Detailed Description

This class encapsulates a simple interface for the DOGM128 LCD display (https://www.lcd-module.com/eng/pdf/grafik/dogm128e.pdf).

This class maintains an internal framebuffer that can be queried and manipulated using the clear(), putpixel(), getpixel() and invpixel() functions. This internal framebuffer is displayed on the LCD using the flip() function.

Example:

#include "lcd.h"
#include <stdlib.h> //rand()
int main(void)
{
//Set random pixels on LCD
lcd display("/dev/spidev0.0", "/dev/gpiochip0", 42, 43);
while(1)
{
display.invpixel(rand()%display::width, rand()%display::height);
display.flip();
}
}
