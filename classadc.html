Detailed Description

This class encapsulates a simple interface for the four input MCP3004 ADC (https://www.microchip.com/wwwproducts/en/MCP3004) connected over a SPI bus.

This class returns a floating point output between 0.0 and 1.0 where 0.0 represents GND and 1.0 represents VCC.

Example:

#include "lcd.h"
#include <unistd.h> //sleep()
#include <cout> //std::cout
int main(void)
{
//Read ADC values every second
adc in("/dev/spidev0.0");
while(1)
{
std::cout << "Values: " <<
in(adc::CH0) << ' '
in(adc::CH1) << ' '
in(adc::CH2) << ' '
in(adc::CH3) << '\n';
sleep(1);
}
}
