Detailed Description

This class encapsulates a simple interface for a debounced GPIO input, ensuring an input is active for a certain amount of time before registering a input.

This class inherits form the gpioin class.

Example:

#include "debounce.h"
#include <iostream> //std::cout
int main(void)
{
debounce in("/dev/gpiochip0", 42, false, 20, "Test Input");
while(1)
{
if(in)
std::cout << "PRESSED\n";
}
}
